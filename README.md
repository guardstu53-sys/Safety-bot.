# Safety-bot.

## What for?
This bot is for safety and security. 

## Where will it get used?
At Rokas community, and maybe other servers. 

## What can you do with this bot?
Install for security and safety. 
It will automatically detect if a user sends an inappropriate message or is spamming/promoting.  It will block the message and give that user an instant timeout/warning. Depending on how severe it is, the mute/warning cannot be appealed. 
Some words will automatically be assigned to ban, warn, kick or mute people! 


Commands:
# ✓ ✕ …
* = required

-- Public --
… Ping - Ping Pong! (silent)
✕ Mywarns - Shows all your warnings, mutes, kicks and bans!! (silent)


-- Mod only -- Select role, any role higher than that will also gain access
✕ SeeStrikes - /Seestrikes {User*} - Moderators can see a user's strike. 
✕ Mute - /Mute {User*} {duration} {reason*} - Moderators can mute a user, and can select a duration. (If no duration is set, it will be automatically depending on the reason.)
✕ Warn - /Warn {User*} {Reason*}

-- Admin only -- Select role, Also has mod perms. 
✕ Unverify - /unverify {User*} {Reason} - Unverifies the selected user. Has to re-verify to gain access.
✕ Ban - /Ban {User*} {Duration*} {Reason*} - Bans a user from the server, Duration is set for how long.
